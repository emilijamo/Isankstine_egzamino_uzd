# Išankstinė egzamino užduotis

*Aprašymas*

Ši užduotis skirta patyrinėti, kad tokio tipo užduotims asociatyvūs konteineriai, t.y. tokie, kurie elementų į konteinerį įrašymo metu juos "surūšiuoja" yra labiau tinkami ir efektyvesni lyginant su sekos konteineriais.

*Programos veiksmingumas*

Programos rezultatas yra 3 tekstiniai failai - žodžių dažnių lentelė, cross-reference tipo lentelė, URL adresų sąrašas.

*Veiksmingumų aprašymai*
Lengvesniam įsivaizdavimui, kaip veikia programa, pateikiami rezultatai, naudojant tokį 'input.txt' failą

1. Žodžių dažnių lentelė

Nurodomi žodžiai, pasikartojantys daugiau nei vieną kartą, bei šalia nurodomas jų dažnis
<img width="505" height="172" alt="image" src="https://github.com/user-attachments/assets/2bba35c8-fb8e-49ff-9f50-f1cc4372538b" />


2. Cross-reference lentelė
   
Lentelėje nurodyta, kurioje teksto vietoje (kurioje(-iose) teksto eilutėse) kiekvienas iš daugiau negu vieną kartą pasikartojantis žodis buvo paminėtas.

<img width="616" height="168" alt="image" src="https://github.com/user-attachments/assets/1145f90b-ac87-4a79-8795-2bdd37abffb3" />

3. URL sąrašas

Iš teksto išrenkami URL'ai, pateikti "pilnu" https://www.vu.lt/, http://www.vu.lt ar "sutrumpintu" pavidalu: www.vu.lt, vu.lt, saitas.xyz, kurių URL galūnė yra 'url_galunes.txt' faile (tai yra https://data.iana.org/TLD/tlds-alpha-by-domain.txt nukopijuotas URL sąrašas)
